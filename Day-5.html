Linux Commands

1. ls – List Directory Contents

Description: Lists files and directories in the current working directory.

Syntax:

ls

Example:

ls -l    # Detailed list




ls -a    # Show hidden files

2. pwd – Print Working Directory

Description: Displays the current directory path.

Syntax:

pwd

Output :

3. mkdir – Make Directory

Description: Creates a new directory.

Syntax:

Output :

4. cd – Change Directory

Description: Navigate between directories.

Syntax:

cd <directory_path>

Example:

5. rmdir – Remove Empty Directory

Description: Deletes an empty directory.

Syntax:

Output :

 

6. cp – Copy Files and Directories

Description: Copies files or directories.

Syntax:

cp <source> <destination>

Example:

7. mv – Move or Rename Files

Description: Moves or renames files and directories.

Syntax:

mv <old_name> <new_name>

Example:

8. rm – Remove Files and Directories

Description: Deletes files or directories.

Syntax:

rm <file_name>

Description: Shows system information.

Syntax:

14. clear – Clear Terminal

Output :

 

15. ps – View Running Processes

Description: Displays currently running processes.

Syntax:

ps

ps aux    # Detailed list of all processes

Output :

ps



ps aux    # Detailed list of all processes

 

16. man – Manual Pages

Description: Displays the manual (help guide) for commands.

Syntax:

man <command>

Example:

man ls

17. grep – Search Text

Description: Searches for specific patterns in a file or output.

Syntax:

Example:

18. echo – Print Text

Description: Displays text or variables.

Syntax:
