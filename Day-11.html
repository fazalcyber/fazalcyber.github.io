 What is IPv4?

IPv4 (Internet Protocol version 4) is the fourth version of the Internet Protocol and the most widely used. It is a connectionless protocol used to identify devices on a network through an addressing system.

�� Key Features:

· 32-bit address format.

· Represented in dotted-decimal notation (e.g., 192.168.0.1).

· Provides around 4.3 billion unique addresses.

· Uses classes and subnetting to organize and manage IP space.

 

IPv4 Address Structure

An IPv4 address is divided into four octets, each 8 bits long:

Binary:     11000000.10101000.00000000.00000001

Decimal:      192     .  168   .   0   .   1

Each octet ranges from 0 to 255.

�� What is Subnetting?

Subnetting is the process of dividing a large network into smaller logical sub-networks (subnets). It helps improve routing efficiency, security, and address management.

�� Subnet Mask
