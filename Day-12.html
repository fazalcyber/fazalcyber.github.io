What is a Gateway?

A Gateway is a network node that acts as an access point to another network, typically used to connect a local network to the internet.

 Key Functions:

· Connects internal network (LAN) to external networks (WAN/Internet).

· Acts as a translator between different protocols.

· Routes traffic from source to destination across networks.

 Common Gateway Example:

· A home router serves as the default gateway for all devices in your home.

· If your IP address is 192.168.1.5, your default gateway is usually 192.168.1.1.

 Default Gateway

A default gateway is used when no other route matches the destination IP.

 Example:

IP Address:       192.168.1.10

Subnet Mask:      255.255.255.0

Default Gateway:  192.168.1.1

Here, all traffic going outside 192.168.1.x is sent to the gateway at 192.168.1.1.

IPv6 Address Format

An IPv6 address is written in eight 16-bit hexadecimal blocks, separated by colons (:):

Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Can be shortened using:

· Leading zero suppression: 2001:db8:85a3::8a2e:370:7334

· :: for consecutive zero blocks (only once per address)

 IPv6 Gateway

In IPv6 networks, the default gateway is a link-local address (e.g., fe80::1), often discovered automatically using Router Advertisement (RA).

Example: 

IPv6 Address:     2001:db8::100

Default Gateway:  fe80::1
